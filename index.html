🚀 دليل شامل لتعلم HTML بالتفصيل 🚀

📌 1. ما هي HTML؟

HTML (HyperText Markup Language) هي لغة ترميز تُستخدم لإنشاء صفحات الويب، حيث تحدد هيكلة المحتوى باستخدام وسوم معينة.

📌 2. الهيكل الأساسي لصفحة HTML

<!DOCTYPE html> <html> <head> <title>عنوان الصفحة</title> </head> <body> محتوى الصفحة هنا </body> </html>

📌 3. الوسوم الأساسية في HTML

  • <h1> - <h6>: تُستخدم للعناوين.
  • <p>: لإنشاء الفقرات النصية.
  • <a href="URL">: لإنشاء الروابط.
  • <img src="image.jpg" alt="وصف الصورة">: لإضافة الصور.
  • <ul> و <ol>: لإنشاء القوائم.
  • <table>: لإنشاء الجداول.
  • <form>: لإنشاء النماذج.
  • <div>: يستخدم كحاوية لتجميع المحتوى.

📌 4. العناصر الدلالية في HTML5

تمت إضافة عناصر دلالية جديدة في HTML5 تساعد على تحسين SEO وتجربة المستخدم:

  • <header>: لتحديد رأس الصفحة.
  • <nav>: لقوائم التنقل.
  • <article>: لمحتوى مستقل.
  • <section>: لأقسام المحتوى.
  • <footer>: لتذييل الصفحة.

📌 5. الجداول في HTML

<table border="1"> <tr> <th>الاسم</th> <th>العمر</th> </tr> <tr> <td>أحمد</td> <td>25</td> </tr> </table>

📌 6. القوائم في HTML

<ul> <li>عنصر 1</li> <li>عنصر 2</li> </ul>

📌 7. النماذج في HTML

<form> <label for="name">الاسم:</label> <input type="text" id="name" name="name" required> <button type="submit">إرسال</button> </form>

📌 8. الوسائط المتعددة في HTML

إضافة فيديوهات وصوتيات بسهولة:

<video controls> <source src="video.mp4" type="video/mp4"> </video> <audio controls> <source src="audio.mp3" type="audio/mpeg"> </audio>
